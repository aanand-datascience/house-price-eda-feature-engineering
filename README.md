## Project Overview

This project performs **Complete Exploratory Data Analysis (EDA)** and **Feature Engineering** on the House Price train dataset.  
The objective is to understand the dataset, analyze important features, handle missing values, detect outliers, and prepare the data for machine learning models.

---

## Dataset Information

The dataset contains information about residential homes and their selling prices.

**Target Variable**
- SalePrice

**Important Features**
- LotArea
- OverallQual
- YearBuilt
- TotalBsmtSF
- GrLivArea
